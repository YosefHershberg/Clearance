# Clearance — Knowledge Vault

Knowledge graph for the Clearance fullstack app. Two submodules: a React/Vite client and a Node/Express server, sharing a Postgres (Prisma) backend.

> [!info] Current state
> The codebase is an early scaffold. Most domain layers (services, data-access, schemas, webhooks, routes, Prisma models, client pages) are stubs. Notes mark these clearly so the graph reflects today's reality, not future intent.
