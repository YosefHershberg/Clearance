# shadcn/ui

Copy-into-your-repo React component recipes. Each component is real source code under `src/components/ui/`, not a black-box dependency. In Clearance the primitives sit on top of [Base UI](https://base-ui.com), styled with Tailwind v4 and `class-variance-authority`.

Components present today:
- [[Button]]
- [[DropdownMenu]]

Class merging is done by [[cn]].
